Overview

This project is a sophisticated implementation of the MFRC522 RFID module paired with the ESP8266 microcontroller. While many RFID projects rely on hardcoded UIDs within the firmware, this solution introduces a dynamic management system.

The system features a web-based dashboard that allows you to:

  • Monitor access logs in real-time.
  • Register or remove users by assigning names and roles (e.g., Manager or Staff).
  • Toggle access permissions instantly without deleting user data.
To enhance the hardware experience, an SSD1306 OLED display provides immediate visual feedback, showing the cardholder's name and authorization status upon scanning.

The frontend is built using Vue and Vite, with both the source code and compiled assets included to allow for easy customization.

Enjoy modifying the project!
